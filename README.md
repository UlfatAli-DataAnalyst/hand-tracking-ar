# Hand Tracking AR Lines 🖐️✨

Real-time hand tracking project that draws colorful AR lines
between fingertips using Python and MediaPipe.

## Demo
Show both hands in front of webcam — colorful lines connect
all fingertips across both hands in real time!

## Technologies Used
- Python 3.14
- MediaPipe
- OpenCV

## How to Run

Install libraries:
pip install mediapipe opencv-python

Run the project:
python main.py

## Features
- Detects both hands simultaneously
- Colorful lines connect all fingertips
- Lines cross between both hands
- Real-time webcam tracking
- Works on Windows with HP laptop
